# 王者荣耀过渡视频Pipeline项目

## 项目概述

这是一个专门为王者荣耀（Honor of Kings）游戏内容创作者设计的视频处理pipeline工具集。项目包含三个核心功能模块，帮助用户从语音素材获取到视频内容生成的完整工作流程。

## 功能模块

### 1. 语音包下载器 (`download_voice.py`)

**功能描述：**
- 从王者荣耀官方语音库自动下载指定英雄的语音包
- 支持英雄名称搜索和模糊匹配
- 批量下载所有皮肤对应的语音文件
- 自动分类保存到对应英雄目录

**主要特性：**
- 智能英雄识别和语音数据提取
- 多线程安全下载，支持断点续传
- 自动文件命名和分类管理
- 交互式命令行界面

### 2. 图片过渡描述词生成器 (`generate_transition_descripition.py`)

**功能描述：**
- 调用豆包API，根据两张图片生成过渡描述词
- 支持多种图片格式（JPG、PNG、GIF、BMP、WebP）
- 批量处理多对图片，自动生成描述文本

**主要特性：**
- 智能图片编码和API调用
- 可自定义生成提示词和参数
- 批量处理支持，自动文件管理
- 完整的错误处理和日志记录

### 3. 水印移除工具 (`remove_watermark.py`)

**功能描述：**
- 自动检测并移除图片右下角的水印
- 提供三种移除方法：修复、模糊、内容感知填充
- 支持单文件处理和批量处理

**主要特性：**
- 智能水印区域检测算法
- 多种移除技术可选，适应不同场景
- 批量处理支持，自动文件命名
- 可调节水印检测区域比例
